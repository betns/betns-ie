{{- /* Image render hook with a small variant vocabulary. Editors opt into a layout by appending ``#token`` to the image URL: ![alt](src) plain inline image ![alt](src#banner) full-width banner across the column ![alt](src#strike) right-floated 350px hero (homepage style) ![alt](src#icon) inline icon (sits beside text in lists/prose) ![alt](src#thumb) left-floated 220px thumb that text wraps around ![alt](src#center) block-centered with optional caption (Title) Path rewriting (e.g. ``/uploads/foo.jpg`` → ``/betns-ie/uploads/foo.jpg``) is handled site-wide by ``canonifyURLs = true`` in ``hugo.toml``; this hook deliberately leaves the URL untouched so the two don't fight. */ -}} {{- $dest := .Destination -}} {{- $variant := "" -}} {{- if strings.Contains $dest "#" -}} {{- $parts := split $dest "#" -}} {{- $dest = index $parts 0 -}} {{- $variant = lower (index $parts 1) -}} {{- end -}} {{- $alt := .PlainText -}} {{- $title := .Title -}} {{- if eq $variant "icon" -}} {{- else if $variant -}} {{- else -}} {{- end -}}
